Livsstilsfaktorer och studieresultat - Vilka livsstilsfaktorer påverkar studieresultaten för studenter?

Författare
Ebba Geldern
Ida Gerdin
Linnéa Jones
Linnéa Lärka

Datasetet kommer ifrån en studie som undersöker livsstilsfaktorer hos studenter i åldrarna 17-24. Studien innefattade 1000 observationer och innehöll 16 variabler. Livsstilsfaktorerna vi har undersökt är sociala medier, sömn, psykisk hälsa och träning och om det har någon påverkan på studieresultatet hos studenter beroende på kön. Detta är en självskattningsstudie.

Frågeställning: Vilka livsstilsfaktorer påverkar studenters studieresultat baserat på kön?

ggplotly(plot)


Detta spridningsdiagram visar sambandet mellan antal timmar på sociala medier och studieresultat, med kön som färgkodning. Alla tre trendlinjer lutar svagt nedåt, vilket tyder på ett negativt samband mellan tid på sociala medier och studieresultat. Ju fler timmar på sociala medier, desto lägre tenderar resultatet att bli.

Summering

library(patchwork)

long_df <- student_habits_performance |>
  pivot_longer(
    cols = c(social_media_hours, sleep_hours, mental_health_rating, exercise_frequency),
    names_to = "variable",
    values_to = "value"
  ) |>
  mutate(
    variable = recode(variable,
                      "social_media_hours" = "Tid på sociala medier (Timmar)",
                      "sleep_hours" = "Sömn (Timmar)",
                      "mental_health_rating" = "Psykisk hälsa (1-10)",
                      "exercise_frequency" = "Träningspass (Vecka)")
  )

make_plot <- function(var_name, x_label) {
  ggplot(filter(long_df, variable == x_label), aes(x = value, y = exam_score, color = gender)) +
    geom_point(alpha = 0.6) +
    geom_smooth(method = "lm", se = FALSE) +
    scale_color_manual(values = c("Female" = "#8ecae6", "Male" = "#023047", "Other" = "#fb8500")) +
    labs(
      title = x_label,
      x = x_label,
      y = "Studieresultat (poäng)",
      color = "Kön"
    ) +
    theme_minimal()
}

labels <- c(
  "Tid på sociala medier (Timmar)",
  "Sömn (Timmar)",
  "Psykisk hälsa (1-10)",
  "Träningspass (Vecka)"
)

plots <- lapply(labels, function(lbl) make_plot("variable", lbl))

wrap_plots(plots, ncol = 2) +
  plot_annotation(title = "Samband mellan olika faktorer och studieresultat")


Resultaten visar att sociala medier är den livsstilsfaktor som i störst utsträckning påverkar studieresultaten negativt. Ju mer tid som spenderas på sociala medier, desto lägre tenderar betygen att bli. Ett mönster som är tydligt oavsett kön.

Vi ser ett tydligt positivt samband mellan psykisk hälsa och studieresultat. Ju bättre studenten mår psykiskt, desto högre tenderar betygen att vara, oavsett kön.

För träning visar resultatet ett svagt positivt samband med studieresultat. Studenter som tränar mer tenderar att prestera något bättre, men effekten är liten och variationen inom grupperna är stor.

Även för sömn ser vi ett svagt positivt samband. Fler timmars sömn hänger samman med något högre studieresultat. Sambandet är likartat för alla kön, men även här finns stor variation i resultaten, vilket tyder på att sömn är en av flera faktorer som påverkar betygen.

Även om vi inte har analyserat sambanden mellan livsstilsfaktorerna, kan det vara så att de påverkar varandra och tillsammans bidrar till studieresultaten. Resultaten väcker därmed frågor om hur livsstilsvanor samspelar och gemensamt påverkar den studieresultaten.
